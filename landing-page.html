AIを活用した文章修正アプリ

テキストを入力し、指示を書くだけで、AIが文章を修正してくれます。文字の置き換え、削除、追加など、様々な修正に対応。

今すぐインストール

主な機能

🤖

AI駆動の文章修正

Claude AIを活用して、自然で正確な文章修正を実現します。

⚡

簡単操作

テキストを入力し、指示を書くだけのシンプルな操作で文章を修正できます。

📋

ワンクリックコピー

修正された文章をワンクリックでクリップボードにコピーできます。

🎯

多様な修正対応

文字の置き換え、削除、追加など、様々な修正ニーズに対応します。

💻

モダンなUI

React + TypeScriptで構築された、美しく使いやすいインターフェース。

🔒

セキュア

Claude APIを使用した安全で信頼性の高い文章修正サービス。

使い方

1

アプリを起動

ショートカットキーでCursor miniを呼び出します

2

テキストを入力

修正したい文章をテキスト挿入欄に入力します

3

指示を書く

チャット欄に修正の指示を入力します

4

AIが修正

AIが指示通りに文章を修正します

5

結果をコピー

修正された文章をコピーボタンでクリップボードにコピーします

ユースケース

❶ 文字の置き換え

入力テキスト: 「デュオリンゴのスマート解説紹介パートの部分共有になります。デュオリンゴのリリー通話パートの部分共有になります。」

指示: 「デュオリンゴをDuolingoに変更」

結果: 「Duolingoのスマート解説紹介パートの部分共有になります。Duolingoのリリー通話パートの部分共有になります。」

修正したいテキスト: "デュオリンゴのスマート解説..." 修正の指示: "デュオリンゴをDuolingoに変更" ↓ 修正結果: "Duolingoのスマート解説..."

❷ 文字の削除

入力テキスト: 全角スペースが含まれたリスト

指示: 「全角スペースをなくして」

結果: 全角スペースが削除されたクリーンなリスト

修正したいテキスト: "　　・Instagram広告..." 修正の指示: "全角スペースをなくして" ↓ 修正結果: "・Instagram広告..."

❸ 文字の追加

入力テキスト: プレーンテキストのリスト

指示: 「文頭に・を追加して」

結果: 各項目に・が追加されたリスト

修正したいテキスト: "Instagram広告キャンペーン..." 修正の指示: "文頭に・を追加して" ↓ 修正結果: "・Instagram広告キャンペーン..."

インストール方法

1
リポジトリをクローン
git clone https://github.com/your-username/cursor-mini.git cd cursor-mini
2
依存関係をインストール
npm install
3
環境変数を設定
cp env.example .env

.envファイルを編集して、Claude APIキーを設定してください

4
開発サーバーを起動
npm run dev

ブラウザで http://localhost:3000 にアクセスしてください
